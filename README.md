# Sacred Scriptures Bible App 📖

A beautiful, modern Bible application built with HTML, CSS, and JavaScript. This Progressive Web App (PWA) works offline and can be installed on any device.

## Features ✨

- 📚 Complete books of the Bible (66 books)
- 🔍 Search for specific verses
- 🌙 Dark/Light mode toggle
- 🔖 Bookmark favorite verses
- 📅 Daily verse
- 📱 Installable as a PWA
- 💾 Offline support
- 🎨 Modern, responsive design

## Installation 📲

### On GitHub Pages
1. Fork this repository
2. Go to Settings > Pages
3. Select "main" branch as source
4. Your app will be live at `https://yourusername.github.io/bible-app`

### Local Development
```bash
# Clone the repository
git clone https://github.com/yourusername/bible-app.git

# Navigate to the folder
cd bible-app

# Open with live server or simply open index.html
